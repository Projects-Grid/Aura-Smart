# 🌌 AURA - Smart Dashboard System

**Aura** هو نظام واجهة مستخدم متطور يعتمد على مفهوم "الهالة الرقمية"، حيث يربط بين تقنيات الويب الحديثة وأجهزة إنترنت الأشياء (IoT).

## 🚀 المميزات
- **الربط الحيوي:** اتصال حقيقي بالساعات الذكية عبر Web Bluetooth API.
- **تصميم عصري:** واجهات Glassmorphism مع تأثيرات نيونية.
- **نمط المحاكاة:** نظام مدمج لعرض البيانات الحيوية في حال عدم توفر عتاد حقيقي.

## 🛠 التقنيات المستخدمة
- HTML5 / CSS3 (Advanced Animations)
- JavaScript (ES6+)
- FontAwesome Icons

## 📸 عرض المشروع
يمكنك فتح ملف `[login.html](https://projects-grid.github.io/Aura-Smart/login.html)` للبدء في تجربة النظام.
